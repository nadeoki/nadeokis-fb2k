# nadeokis-foobar2k (2.0) 64-bit
a custom theme for a black-red style

![preview](https://i.imgur.com/ePqrBTr.png)

## Installation

Go to [releases](https://github.com/nadeoki/nadeokis-fb2k/releases)
- and download the latest version
- make sure you have foobar2000 2.0 (64-bit) portable installed download it [here](https://www.foobar2000.org/windows)
- unzip the release into your portable Foobar2000 directory
- open foobar2000, got to Display > Default Interface > Import Theme and select "nadeokis-fb2k-theme.fth"
- check dependencies if something doesn't work.
